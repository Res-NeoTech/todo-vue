# Vue To-Do App

I created this project intending to learn the basics of the **Vue.js** framework. Just a simple to-do app with some cool features.

## Guides

### Install the dependencies
```bash
npm install
```

### Run the dev server
```bash
npm run dev
```

### Run the production build
```bash
npm run build
```
